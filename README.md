
dwm - dynamic window manager
============================

my patched version of dwm

- `git remote add source git://git.suckless.org/dwm`

- `git pull --rebase source main`

- `wget https://dwm.suckless.org/patches/patch.diff`

- `patch -p1 < patch.diff`

- `make`

- `make clean install`

source:
